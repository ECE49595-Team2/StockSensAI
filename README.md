# StockSensAI
Awesome trade simulation app that assists in making stock market decisions

## Features
- Buy and sell stocks using fake money on the IEX exchange.
- Practice making sound investing decisions with the help of AI news analysis.
- Chat with a built-in assistant for any finanacial questions you may have.
- Automatically execute trades even when you're not using the app based on predefined algorithms.
- Get used to the environment of a real stock trading app with an intuitive user interface.
- Manage multiple portfolios to try out different combinations of stocks and strategies. See what works and what fails without financial consequences.

## Getting started
Users can go here https://stocksensai.com/
Developers can build each container within the frontend, backend/algorithmic-trader, and backend/llm-api directories. You will also have to start a local instance of couchdb using the premade docker container. Then modify the docker-compose.yaml to use these containers and run docker-compose up to run locally.
